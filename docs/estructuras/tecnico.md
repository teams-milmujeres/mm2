# ¿Qué hace?

    La aplicación permite a usuarios beneficiarios de Mil Mujeres consultar el
    estado de su caso legal, revisar pagos y depósitos, acceder a eventos e
    información relevante de la organización, y visualizar su perfil personal. Todo
    esto mediante una conexión segura con el sistema interno Teams.

# ¿Donde se instala?

    Se instala en dispositivos móviles con sistema operativo Android o iOS. Está
    disponible a través de la Google Play Store y la Apple App Store.

# ¿Que lenguaje usa?

    El frontend está desarrollado en Dart, utilizando el framework Flutter. La API
    backend (Teams) está desarrollada en PHP usando el framework Laravel.

# ¿Para que sirve?

    Sirve como un canal de comunicación directo y seguro entre la organización Mil
    Mujeres y sus beneficiarios, permitiendo el acceso a información legal
    personalizada, seguimiento de casos y participación en actividades.

# Donde se publica´

    Se publica en:
        ● Google Play Store (para Android)
        ● Apple App Store (para iOS)

    Nota: el acceso requiere una cuenta proporcionada por el equipo de Mil
    Mujeres.
